Program 3 - Generate numbers and sort with generate.cpp and sort.cpp
- Demonstrate the ability to implement and compare sorting algorithms
- Generate large datasets of random integers using C++ and file I/O
- Writing bubble sort function operating on a vector
- Manage command-line arguments for program output and testing
- Use Linux shell commands and scripts to automate sorting, timing
- Comparing the performance of a custom sort with a system's built in sort
- Practice good software engineering habits, including code documentation

How to run program 3. 

Navigate to folder
cd Program3

Compile both programs:
g++ -o generate generate.cpp
g++ -o mysort mysort.cpp

Run the script and save output to a log file:
rm -f sortrace.log
./sortrace.sh >> sortrace.log 2>&1

Check the results:
tail -n 40 sortrace.log

Notes:
- The script creates numbers.dat, sorts with system sort and mysort, and checks that mysort.out is sorted.
- Large input sizes may take a long time due to bubble sort being used and compared against system sort.
