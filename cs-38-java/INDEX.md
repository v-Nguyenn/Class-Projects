## üìù Notes

- Files ending in `-1.java` are typically example files from the textbook
- Files with `YName` are templates (replace with your name)
- `.ctxt` files are BlueJ metadata (can be ignored in VS Code)
- `package.bluej` files are BlueJ project files (can be ignored in VS Code)
- Many classes are in the **default package** (no `package` statement), so they must be compiled from the workspace root
